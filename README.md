# eresidency

A Claude Code skill that guides remote founders through Estonian e-Residency, company formation, banking, accounting, and getting paid with stablecoins.

Built from the real experience of [Panche Isajeski](https://linkedin.com/in/pancheisajeski) — a founder from Macedonia who went through the full process twice and now runs Aristokrates OÜ, the creator of [AgentaOS](https://agentaos.ai), from Estonia.

## Install

```bash
npx skills add AgentaOS/eresidency@eresidency
```

## What it covers

1. **e-Residency application** — cost, requirements, timeline, pickup locations by region
2. **Company formation** — OÜ registration, service providers compared (Xolo, Companio, 1Office, Enty.io)
3. **Accounting** — what's required, costs, bundled vs standalone options
4. **Banking** — Wise, LHV, Revolut, honest pros/cons of each
5. **Getting paid with stablecoins** — payment vs settlement currency, MiCA, processors (AgentaOS, Request Finance, Swapin, Stripe)
6. **Estonian tax overview** — 0% retained, 22% distributed, VAT at 24% (current 2026 rates)
7. **Hard-won tips** — share issuance, buybacks, VAT registration, fundraising signing, accounting automation

## Who this is for

- Remote founders who want an EU company
- Freelancers looking for a legitimate EU business structure
- Non-EU citizens who need a way to operate in Europe
- Anyone stuck in banking limbo
